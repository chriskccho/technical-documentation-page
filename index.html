Introduction

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Its high-level built in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development, as well as for use as scripting or glue language to connect existing components together. Python's simple, easy to learn syntax emphasizes readability and therefore reduces the cost of program maintenance. Python supports modules and packages, which encourages program modularity and code re-use. The python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.

Hello World

To get started with writing Python, open an IDE and write your first "Hello World" Python code:

print("Hello World")

The print() statement outputs the string inside to the console. The above statement will output the string "Hello World". You can also pass other data types such as integer to the print statement.

Variables

Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

x = 5
x = "hello"

Note that you do not require any ; at the end of each statement. You can also change the variable to any other data types and it will switch based on your value.

Data Types

Python has many different data types for use, the following list are all the data types in these categories:

  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview

The above data types can be used for many different applications and it is a fundamental component of the Python programming language.

  • Text type str can be used to hold text, such as x = "Hello World".
  • Numeric types can hold numbers, int for integers flaot for decimal numbers and complex for complex numbers.
  • Sequence types are essentially arrays but each data type of list, tuple, and range are used for different purposes.
  • Mapping type dict is used to access values via a key, the key will point to the value corresponding to it and it is a very useful data type.
  • Set types such as set and frozenset are similar to the sequence types and mapping types but the key difference is that they do not allow duplicate values.
  • Boolean type bool allows two values: either True or False.
  • Binary types are used for working with binary data, byte and bytearrays are used to manipulate binary data, and are supported by buffer protocol, named memoryview.

Conditional Statements

Conditional statements are used to simply use a condition that the code must satisfy in order to execute.

x = 1
if x == 1:
    x += 1

The above code will run as follows:

  • The variable x is set to 1.
  • The if statement checks if the x value is equal to 1.
  • Since the condition is satisfied, the code in the indented section executes and x = 2 after the code executes.

It is important to note since python does not use any ; or {} like other programming languages, it is important to indent the contents of the if statement as this is how Python recognizes it.

You can also have multiple conditions for conditional statements:

if x == 1:
    x += 1
elif x == 2:
    x += 2
else: 
    x += 3

The above code will run as follows:

  • The first conditional statement if x == 1 will check if x is equal to 1.
  • If not, it will check the statement elif x == 2 if x is equal to 2.
  • If both statements are not satified, the code within the else statement will execute.

Loops

Loops can be used to execute code multiple times while the condition is satisfied.

x = 1
while x < 3:
    x += 1

The above code will run as follows:

  • The section of code within the while statement will run as long as the condition specified is satisfied.
  • With x = 1, this is less than 3, hence the code will add 1 to the x value.
  • Now with x = 2, the code is executed again, as 2 < 3.
  • Now with x = 3, 3 is not less than 3, thus the loop stops.

There are other ways to create loops into your code, the below being another example:

x = 1
for i in range(6):
    x += 1

The above code has i in it, which increments each time the loop executes, the loop will execute 6 times as specified in range(6).

Functions

Functions are a way to store and re-use a set of code.

def print_something(x):
    print(x)
x = "hello"
print_something(x)

The above code is an example of a function that prints whatever the user passes to it. The function is created with def print_something(): where print_something is the name of the function. Hence, this code will print to console the string "hello"

You can also pass multiple parameters and ask the function to return the results so you can do further things with it:

def add_together(x, y):
    return x + y
x = 3
y = 2
z = add_together(x, y)

The above code has a function add_together that takes two parameters and returns the sum. The variable z stores the results of the function, hence the value of z will be 5.

Classes

Classes are the core to OOP, it allows you to encapsulate different variables and functions inside what is called an "Object" (an instance of a class) and re-use the class over and over again. Essentially, classes are a template for an object in which upon creating it, you can create multiple objects that utilizes the same attributes and methods inside.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

p1 = Person("John", 36)

print(p1.name)
print(p1.age)

You can create a class by using Class classname: and the __init__() function is executed when the class is being initiated. When the code p1 = Person("John", 36) executes, it assigns the passed parameters into the object's attributes. In this example, the name and age are passed as "John" and 36 respectively and assigned as such in the object.

You can access the attributes you have created inside the object by using the . operator. If i want to access the name of an object called p1 I can use the statement p1.name. In the case of the above code, the printed statement of p1.name will be "John".

Methods

Functions inside a class are called methods, they are useful since it abstracts the function away from the code and keeps it inside of the object.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def myfunc(self):
        print("Hello my name is " + self.name)

p1 = Person("John", 36)
p1.myfunc()

The method myfunc() takes its attribute name and prints out the statement as specified above. Hence when the object p1 is initialized with the name "John" the statement p1.myfunc() will output the string "Hello my name is John".
